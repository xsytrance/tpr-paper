\# Technological Phylogenetics with Selection Pressure Annotation



This repository contains a research paper proposing a framework for representing the evolution of technological artifacts using commit-like structures and selection pressure annotation.



The framework captures both:

\- objective pressures (performance, cost, engineering constraints)

\- subjective pressures (perception, safety, cultural acceptance)

\## 🧪 Pilot Study



Located in `/pilot/tpr-pilot-clean`



A structured reconstruction of the safety bicycle lineage (1876–1892), demonstrating:



\- commit-based artifact evolution

\- pressure annotation (objective + subjective)

\- evidence-backed transitions
