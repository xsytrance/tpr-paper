# TPR Pilot — Clean Artifact Package

**Purpose:** Publication-ready supplementary material for the TPR (Technological Phylogenetic Repository) framework demonstration.

**Contents:**
- `source_register_clean.md` — 15-source evidence table with roles and reliability
- `commit_objects_clean.json` — Structured TPR commit objects (4 commits + tire subcomponent)
- `lineage_narrative_clean.md` — Concise evolutionary narrative
- `pressure_comparison_clean.md` — Objective-only vs hybrid analysis
- `validation_clean.md` — Claim-strength assessment (strong/medium/weak)
- `visual_clean.md` — Lineage tree and pressure mapping
- `worked_example_transition.md` — Paper-ready C→D transition write-up
- `pressure_table.md` — Compact pressure evidence table
- `core_transition.md` — Justification for selecting C→D as the core example
- `README.md` — This overview

**Scope:** Safety bicycle, 1876–1892. 4 commits (A–D), 1 subcomponent (tire), 15 sources.

**Key result:** Hybrid pressure annotation adds explanatory value primarily at the pneumatic-tire transition (C→D), where comfort improvements enabled social expansion (women riders) and altered health/safety perceptions.
